== Description ==

Enable full oauth2 authentication for your site. This plugin used league oauth2 server. It's highly secured and most used technology.

There is few steps to create your credentials:

*   First go to `WP OAuth2 Server` menu.
*   Then click `Clients` sub menu.
*   Then click on `Create new client`. It'll popup a modal. Fill up the form and click `save`.
*   Click `show` to see each client credentials, click 'edit' to update and click `remove` button to delete/destroy current client.
